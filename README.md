# Project Installation Guide

## Follow the steps below to install and set up the project:

## Install Git
Download and install Git from the official website:
https://gitforwindows.org

## Open Git (Terminal)
After installation, open Git Bash (the terminal).

## Navigate to a Folder
Use standard terminal commands to move to the directory where you want to place the project, for example:

cd path/to/your/folder

---

## Clone the Repository
In the selected folder, clone the repository using the git clone command.
All project files will be downloaded into this directory.

### Download the Project Files
Download the project from the website (Download Project), if required.

### Import the Database
Download the provided SQL file and import it into your database (export/import the database into the project).

After completing all the steps above, you will be able to successfully use the repository from GitHub.
